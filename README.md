# Rank-Tracker
Rank Tracker App, to get the ranking of your webiste.

# Tech Stack
1. NextJS
2. TailwindCSS
3. BrightData API
4. MongoDB

# Installation Steps
1. Run the "npm i" or "yarn install" command to install all the node_modules.
